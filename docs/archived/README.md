## Document Retention Policy

**Archived documents are retained for:**
- Historical reference and learning
- Audit and compliance purposes
- Understanding project evolution
- Avoiding repeated mistakes

**Documents are archived when:**
- The work they describe is 100% complete
- They have been superseded by newer versions
- They are no longer referenced in active plans
- The strategic direction has changed

**Active documents remain in `/docs/` until:**
- Implementation is 100% complete (then moved to archive)
- The feature/plan is cancelled (then moved to abandoned folder)
- A newer version supersedes them (old version archived)
