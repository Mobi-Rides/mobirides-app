# Mobirides Checklists

This folder contains all project checklists organized by category:

## Folder Structure

```
checklists/
├── README.md              # This file
├── checklist.md          # Master checklist index
├── daily/               # Daily development checklists
├── sprint/              # Sprint execution checklists  
├── release/            # Release preparation checklists
└── deployment/          # Deployment checklists
```

## Categories

### Daily (`daily/`)
- Daily standup checklists
- Code review checklists
- Bug investigation checklists

### Sprint (`sprint/`)
- Sprint planning checklists
- Sprint review/retrospective checklists
- Iteration progress checklists

### Release (`release/`)
- Pre-release validation checklists
- Feature completion checklists
- Production readiness checklists

### Deployment (`deployment/`)
- Deployment verification checklists
- Rollback procedures
- Post-deployment verification

## Usage

Each subfolder contains its own README with specific instructions. The master `checklist.md` in this folder provides a comprehensive index of all available checklists.
