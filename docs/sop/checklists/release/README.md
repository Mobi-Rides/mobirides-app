# Release Checklists

This folder contains checklists for release preparation and validation.

## Available Checklists

| Checklist | Description |
|-----------|-------------|
| [feature-completion.md](feature-completion.md) | Feature completion verification |
| [pre-production.md](pre-production.md) | Pre-production readiness |
| [production-checklist.md](production-checklist.md) | Production release validation |

## Usage

1. **Feature Completion**: When features are ready for testing
2. **Pre-Production**: Before deploying to production
3. **Production Checklist**: Day of release validation

## Release Process

1. Complete feature checklist
2. Verify pre-production readiness
3. Execute production checklist
4. Deploy and verify
