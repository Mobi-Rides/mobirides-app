# Daily Standup Checklist

Use this checklist for daily standup meeting preparation and tracking.

## Before Standup

### Personal Updates
- [ ] Review yesterday's completed tasks
- [ ] Identify blockers and impediments
- [ ] Update Linear issues with progress
- [ ] Check assigned PRs awaiting review

### Team Updates
- [ ] Review team member progress (via Linear)
- [ ] Note any dependencies needed
- [ ] Prepare questions for others

## During Standup

### What to Share
- [ ] What I completed yesterday
- [ ] What I'm working on today
- [ ] Any blockers encountered

### Information to Gather
- [ ] Dependencies from teammates
- [ ] Context for blocked tasks
- [ ] Priority clarifications

## After Standup

### Follow-up Actions
- [ ] Update issue status in Linear
- [ ] Move issues to correct column
- [ ] Schedule any needed syncs
- [ ] Review day's priorities
