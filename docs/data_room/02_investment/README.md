## Documents

| Document | Description |
|----------|-------------|
| [TERM_SHEET_TEMPLATE.md](TERM_SHEET_TEMPLATE.md) | Master term sheet |

| [INVESTMENT_OFFER_LETTER.md](INVESTMENT_OFFER_LETTER.md) | Offer letter to investors |

1. Customize TERM_SHEET for lead investor
2. Use SAFE for each investor
3. Generate offer letter with amounts
4. Include PRO_RATA with SAFE
