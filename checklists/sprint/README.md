# Sprint Checklists

This folder contains checklists for sprint ceremonies and iteration management.

## Available Checklists

| Checklist | Description |
|-----------|-------------|
| [planning.md](planning.md) | Sprint planning preparation |
| [retrospective.md](retrospective.md) | Sprint retrospective checklist |
| [iteration.md](iteration.md) | Iteration progress tracking |

## Usage

1. **Planning**: Use before sprint planning meetings
2. **Retrospective**: Use after each sprint to capture learnings
3. **Iteration**: Use throughout the sprint to track progress

## Sprint Cadence

- Sprint Planning: Start of sprint
- Daily: Iteration tracking during sprint
- Sprint Review + Retro: End of sprint
