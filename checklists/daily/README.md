# Daily Checklists

This folder contains checklists for daily development activities.

## Available Checklists

| Checklist | Description |
|-----------|-------------|
| [standup.md](standup.md) | Daily standup meeting checklist |
| [code-review.md](code-review.md) | Code review checklist |
| [bug-investigation.md](bug-investigation.md) | Bug investigation workflow |

## Usage

1. Open the appropriate checklist before starting the activity
2. Check off items as they are completed
3. Add notes for future reference
4. Update status in Linear where applicable

## Tips

- Use standup checklist before daily meetings
- Keep code review checklist handy during PR reviews
- Bug investigation checklist is useful for systematic debugging
