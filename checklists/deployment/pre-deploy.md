# Pre-Deployment Checklist

Verify all prerequisites before deployment.

## Prerequisites

### Environment
- [ ] Production environment accessible
- [ ] Database accessible
- [ ] API services accessible
- [ ] CDN configured

### Configuration
- [ ] Environment variables set
- [ ] Secrets configured
- [ ] Feature flags set
- [ ] Domain configured

## Code State

### Repository
- [ ] Working directory clean
- [ ] Branch correct
- [ ] Latest pulled
- [ ] No uncommitted changes

### Tests
- [ ] Unit tests passing
- [ ] Integration tests passing
- [ ] Manual tests passing

## Build Verification

### Build Process
- [ ] Clean install
- [ ] Build completes
- [ ] No warnings
- [ ] Output validated

### Assets
- [ ] Static assets built
- [ ] Bundles optimized
- [ ] Assets uploaded

## Database

### Migrations
- [ ] Migrations ready
- [ ] Rollback ready
- [ ] Data backup verified

### Schema
- [ ] Schema validated
- [ ] Indices verified
